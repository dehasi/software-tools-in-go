== Bubble & Shell Sort

They are in the `sorting` module, tested by unit-tests.

== Sorting Text

`Go` can compare strings lexicographically, therefore no additional `cmp` method is needed.
Also, as `Go` array is a pointer to strings, no additional `posbuf` needed.

.inmemsort
----
$ go build  sorttext.go  io.go

$ ./sorttext
zzz
aaa
nnn
aaa
nnn
zzz

----

However, to keep the code close to the original idea, I've created  `posbuf` as array of pointers to `string`.
Then I sort and print `posbuf`. To show that the original input is not changed, I print original input as well.

.modified inmemsort
----
$ go build  sorttext.go  io.go  && ./sorttext
zzz
aaa
nnn
SORTED:
aaa
nnn
zzz
ORIGINAL:
zzz
aaa
nnn
----
